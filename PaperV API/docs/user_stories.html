User Stories

Loads all the stories that has been created by the user.

URL

  1. http://paperv.com/api/user_stories.php

Request Type

  1. GET

Request Params

  1. user_id : User ID
  2. page : page number

Response Format

  1. Success :
    {
    "success" : true,
    "data" :
    [
      {
       "story_id" : 123,
       "title" : "John in Madrid",
       "total_comment" : 6,
       "total_like" : 15,
       "total_repsot" : 5,
       "photo_url" : <STORY PHOTO URL>,
       "user_id" : <STORY OWNER ID>,
       "user_image" : <OWNER PHOTO URL>,
       "user_fullname" : <OWNER FULL NAME>
      }
    ]
    }
  2. Fail :
    {
       "success" : false,
       "msg" : "Failed to load user stories!"
    }
