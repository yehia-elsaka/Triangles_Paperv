Registration Request

URL

  1. http://paperv.com/api/register.php

Request Type

  1. POST

Request Params

  1. user_name : Username
  2. email : User's email
  3. full_name : User's full name
  4. password : User's password

Response Format

  1. Success :
    {
       "success" : true,
       "user_id" : 123,
       "msg" : User registered successfully
    }
  2. Fail :
    {
       "success" : false,
       "msg" : "Username or email already exists!"
    }
